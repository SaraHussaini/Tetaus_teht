# 01. Asennukset

## Tavoite  
Tavoitteena on asentaa **Robot Framework** -automaatiotestaustyökalu ja sen lisäosat.

## Tarvittavat työkalut  
Asenna seuraavat työkalut:  
- Robot Framework  
- Browser Library  
- Requests Library  
- CryptoLibrary  
- Robotidy  

## Asennusohjeet  

### 1. Luo testauskansiot
